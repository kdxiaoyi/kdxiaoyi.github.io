# 起因
看了某个引流号的视频突发奇想做个颜值更高且没有套路的U盘锁屏软件

# 原理
通过检测指定目录的文件所含**私钥**是否与本地**公钥**相吻合。<br>
若不吻合则弹出自绘锁屏页面，并持续进行上述检查，直到吻合。<br>
加密算法：`SHA256`<br>

# 兼容平台
理论上Windows XP→11都支持

为了防止手贱把U盘搞不见了、配置文件乱改一通完全识别不了，**此时可以右键锁屏，来使用Windows锁屏解锁，并且软件自动退出**<br>
